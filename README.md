# Professional Portfolio

A modern, responsive React portfolio website with a unique design template.

## üöÄ How to Run

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm start
   ```

3. **Open your browser:**
   - The portfolio will automatically open at `http://localhost:3000`
   - If it doesn't open automatically, navigate to that URL

### üìù Customization

1. **Update your information:**
   - Open `src/App.js`
   - Modify the `portfolioData` object with your actual resume information:
     - Name, title, contact details
     - Skills, experience, projects
     - Education information

2. **Customize styling:**
   - Edit `src/index.css` to change colors, fonts, or layout
   - The design uses CSS Grid and Flexbox for responsiveness
